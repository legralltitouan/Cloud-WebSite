# ☁️ Angora Cloud

**Angora Cloud** est une interface web moderne et intuitive de gestion de fichiers (Cloud Storage). Ce projet met l'accent sur une expérience utilisateur fluide avec un tableau de bord complet pour visualiser, organiser et gérer vos documents personnels.

---

## 🚀 Fonctionnalités

- **Tableau de Bord Intégral** : Vue d'ensemble sur le stockage utilisé (Documents, Audio, etc.) avec un graphique circulaire dynamique.
- **Navigation Latérale** : Menu structuré pour accéder rapidement à votre Cloud, aux vues générales et aux analyses de données.
- **Gestion de Fichiers** : Interface propre pour lister les dossiers et fichiers récents.
- **Barre de Recherche** : Recherche interactive intégrée pour retrouver vos documents en un clin d'œil.
- **Design Adaptatif** : Interface optimisée pour une utilisation sur desktop (Sidebar fixe et contenu scrollable).

---

## 🛠️ Stack Technique

- **HTML5** : Structure sémantique du dashboard.
- **CSS3** : 
  - Utilisation de **Flexbox** et **CSS Grid** pour une mise en page robuste.
  - Design Dark-mode élégant pour la barre latérale.
  - Effets de survol (hover) sur les cartes et les liens de navigation.
- **JavaScript (Vanilla)** : 
  - Gestion des états actifs dans la navigation.
  - Interaction dynamique avec les cartes de fichiers.
  - Simulation de recherche et mise à jour de l'affichage.
- **FontAwesome** : Bibliothèque d'icônes pour une interface visuelle claire.
